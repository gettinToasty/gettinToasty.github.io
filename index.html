*slackr

*slackr is a team messaging app for teams who want to do anything but work. It is built on a Rails backend using React and Redux to handle dynamic frontend rendering. Some core features include real time messaging, support of the Giphy API, and a full library of Twitter's emojis.

Toasty CSS

Life is beautiful, websites should be too. Toasty CSS is a lightweight, open-source alternative to popular web frameworks like Bootstrap. Built entirely in CSS and SASS, Toasty CSS offers users a quick and painless way to implement modern styling on their webpage.
